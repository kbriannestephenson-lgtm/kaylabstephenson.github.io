# Kayla B. Stephenson — GitHub Pages Portfolio

## Publish on GitHub Pages
1. Upload these files to your GitHub Pages repo (usually: `yourusername.github.io`)
2. In GitHub: Settings → Pages → Source: **Deploy from a branch**
3. Branch: **main** • Folder: **/(root)**
4. Save, then wait for GitHub Pages to build.
